# health-prediction-app
Project Name : health-prediction-app

Overview :The Health Prediction Application is a web-based application developed using Python Flask. It allows users to manage patient health records and generate a health risk prediction based on blood test results.

The application stores patient information including Full Name, Date of Birth, Email Address, Glucose, Haemoglobin, and Cholesterol levels. Based on the entered blood test values, the system automatically generates a health prediction and displays it in the Remarks field.

Technology Stack : 
Backend
Python
Flask

Frontend
HTML
CSS
Bootstrap 5

Database
SQLite

AI/ML Logic
Rule-based health prediction model using blood test values

Features
Add new patient records
View all patient records
Update existing patient records
Delete patient records
Automatic health prediction generation
Input validation for email, date of birth, and numeric values
Persistent data storage using SQLite
Responsive and user-friendly interface
